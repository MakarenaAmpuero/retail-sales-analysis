# Retail Sales Analysis


Este proyecto analiza un dataset de ventas de una tienda retail. El objetivo es limpiar, explorar y visualizar los datos para entender mejor el comportamiento de las ventas.

---

## Tecnologías utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Estructura del proyecto

- Parte I: Carga y exploración inicial de datos  
- Parte II: Limpieza y tratamiento de datos  
- Parte III: Transformación y creación de variables  
- Parte IV: Análisis Exploratorio de Datos (EDA) y visualización  

---

## Parte I: Exploración inicial

Se revisó el dataset con funciones básicas como `head()`, `info()`, `shape` y `describe()` para entender su estructura general y tipos de datos.

---

## Parte II: Limpieza de datos

Se realizaron procesos de limpieza como:
- Revisión de valores nulos
- Corrección de tipos de datos
- Eliminación o tratamiento de datos inconsistentes

---

## Parte III: Transformación de datos

Se crearon nuevas variables para enriquecer el análisis:
- Ingreso calculado
- Ventas
- Ventas normalizadas
- Clasificación de ventas
- Descuento simulado

---

##Parte IV: Análisis Exploratorio (EDA)

Se realizó un análisis exploratorio utilizando estadísticas descriptivas y visualizaciones.

Se aplicaron:
- Histogramas para ver la distribución de variables
- Boxplots para detectar valores atípicos
- Gráficos de líneas para analizar ventas en el tiempo
- Gráficos de dispersión para ver relaciones entre variables

Este análisis permite comprender el comportamiento general de las ventas y detectar patrones importantes.

---

##Conclusión

El análisis permitió entender mejor el comportamiento del dataset, identificar patrones en las ventas y visualizar relaciones entre variables. Además, se detectaron posibles valores atípicos y tendencias en el tiempo.

Este trabajo sirve como base para futuros análisis más avanzados o modelos predictivos.
